## Advent of Code - 2022 - in Ethereum Solidity 


### Getting started 

```shell
$ npm install 
$ npx hardhat compile
```

### Run tests 

```shell
$ npx hardhat test
```


### Solve puzzle
Run task via 

```shell
$ npx hardhat solve-puzzle --day XX (01, 02_1, 02_2, 03, ...)
```
